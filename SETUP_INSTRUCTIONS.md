## 📊 Configuración de Analytics (Opcional)

### Google Analytics 4 (GA4)

1. Obtén tu Measurement ID en [Google Analytics](https://analytics.google.com)
2. Abre `/src/app/utils/analytics.ts`
3. Reemplaza `[TU_ID_AQUI]` con tu ID real (formato: `G-XXXXXXXXXX`)
4. Agrega el script a tu `index.html` (instrucciones en el archivo)

### Meta Pixel (Facebook)

1. Obtén tu Pixel ID en [Facebook Events Manager](https://business.facebook.com/events_manager)
2. Abre `/src/app/utils/analytics.ts`
3. Reemplaza `[TU_ID_AQUI]` con tu Pixel ID
4. Agrega el script a tu `index.html` (instrucciones en el archivo)

### Eventos Trackeados
- ✅ Clics en botón de WhatsApp
- ✅ Cambios de idioma
- ✅ Profundidad de scroll (25%, 50%, 75%, 100%)

## 🌐 Deployment

### Opción 1: Netlify (Recomendado)
1. Conecta tu repositorio Git
2. Build command: `npm run build`
3. Publish directory: `dist`
4. Deploy!

### Opción 2: Vercel
1. Importa tu proyecto
2. Framework preset: Vite
3. Deploy!

### Opción 3: Hosting Tradicional
1. Ejecuta `npm run build`
2. Sube la carpeta `dist` a tu servidor
3. Configura el dominio: `abogadomorales.mx`
